# Smart-Home-Energy-Tracker

Smart-Home-Energy-Tracker is a C++ console application for monitoring and managing home appliance energy consumption. 
Developed as a 3rd semester mini-project, it demonstrates object-oriented programming, file handling, and energy cost calculation.

# Features:

1.Add appliances (Light, Fan, AC, Fridge) with power ratings.

2.Record daily usage hours for appliances.

3.Calculate energy consumption (kWh) and estimated electricity cost.

4.Save/load appliance data (appliances.txt) and log operations (operations.txt).

5.Menu-driven interface for easy interaction.

# Usage:

1.Compile:

g++ -o SmartHomeEnergyTracker main.cpp


2.Run:

./SmartHomeEnergyTracker


3.Menu Options:

* Add Appliance

* Record Usage

* Show All Appliances

* Generate Bill

* Exit
